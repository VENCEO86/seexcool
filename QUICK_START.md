# 🚀 빠른 시작 가이드

## ✅ 프로젝트 완료 상태

**전체 전수 조사 완료!** 모든 기능이 구현되었고 프로덕션 준비 완료 상태입니다.

## 🌐 접속 정보

### 메인 페이지
- **로컬 주소**: http://localhost:3000
- **기능**: 이미지 업로드, 스케일업, 밝기/명암 조절, 다운로드, 클립보드 복사

### 관리자 페이지
- **로컬 주소**: http://localhost:3000/admin
- **비밀번호**: `admin123`
- **기능**: 광고 배너 설정 관리 (상단/하단)

## 📋 실행 방법

### 1. 의존성 설치 (이미 완료됨)
```bash
npm install
```

### 2. 개발 서버 실행
```bash
npm run dev
```

서버가 시작되면 자동으로 브라우저가 열리거나, 수동으로 다음 주소로 접속하세요:
- http://localhost:3000

### 3. 프로덕션 빌드 (선택사항)
```bash
npm run build
npm start
```

## 🎯 주요 기능

### 이미지 편집기
- ✅ 파일 선택 업로드
- ✅ 드래그 앤 드롭
- ✅ Ctrl+V 붙여넣기
- ✅ 모바일 카메라 촬영
- ✅ 스케일업 (1.0x ~ 4.0x)
- ✅ 밝기 조절 (50% ~ 150%)
- ✅ 명암 조절 (50% ~ 150%)
- ✅ PNG 다운로드
- ✅ 클립보드 복사

### 키보드 단축키
- `Ctrl/Cmd + S`: 이미지 다운로드
- `Ctrl/Cmd + Shift + C`: 클립보드 복사
- `Ctrl/Cmd + V`: 이미지 붙여넣기

### 관리자 기능
- 광고 배너 이미지 URL 설정
- 광고 배너 링크 URL 설정
- Alt 텍스트 설정
- JSON 미리보기
- localStorage 기반 저장

## 🔧 문제 해결

### 포트 3000이 이미 사용 중인 경우
Next.js가 자동으로 다른 포트(3001, 3002 등)를 사용합니다.
터미널에 표시된 주소를 확인하세요.

### 서버가 시작되지 않는 경우
```bash
# 포트 확인
netstat -ano | findstr :3000

# 프로세스 종료 후 재시작
npm run dev
```

## 📝 참고사항

- 모든 이미지 처리는 브라우저에서만 이루어집니다 (서버 저장 없음)
- 광고 설정은 localStorage에 저장됩니다
- 최대 이미지 크기: 20MB
- 지원 형식: JPG, PNG, WebP
